========================================
(audio position)
========================================

``(position-new bars beats sixteenths ticks sub_tick)``
   Return a newly-created position as
   ``bars``.\ ``beats``.\ ``sixteenths``.\ ``ticks``.\ ``sub_tick``.


``(position-print pos)``
   Prints the position as ‘bars.beats.sixteenths.ticks.sub_tick‘.
