Common Controls
---------------

Zoom Controls
~~~~~~~~~~~~~

Zoom In
  Zooms in.
Zoom Out
  Zooms out.
Original Size
  Zooms back to the default zoom level.
Best Fit
  Zooms in or out as much as required to show all of
  the events in the timeline.
